> Attributes can be specified in four different ways:
> [Spec Reference](https://html.spec.whatwg.org/multipage/syntax.html#attributes-2)
>
> 1. Empty attribute syntax (e.g. `<p class>`)
> 2. Unquoted attribute value (e.g. `<p class=red>`)
> 3. Single-quoted attribute value (e.g. `<p class='red'>`)
> 4. Double-quoted attribute value (e.g. `<p class="red">`)
>

In this case, the attribute value is unquoted red, which is a valid CSS class name.

Therefore, the color of the text "FullstacksJS" is red.
